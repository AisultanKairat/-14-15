1. GitHub-та кодыңды дайындау • GitHub-та репозиторий жаса. • Қажетті JavaScript кодтарыңды репозиторийге жүкте. • Егер API қолданғың келсе, оған рұқсат беруді ұмытпа. 2. Wix Velo-ны қосу • Wix-те сайт редакторына кір. • Dev Mode (Velo) функциясын қос. • Бұл үшін оң жақ жоғарғы бұрышта Dev Mode қосқышын бас. 3. Wix-те сыртқы файлды қолдану • Public немесе Backend папкаларына жаңа файл қосып, GitHub-тағы кодты көшіріп қоюға болады. • Егер код GitHub-тан API арқылы қосылса, HTTP сұраныстарды қолдану керек: fetch('https://api.github.com/repos/пайдаланушы/репозиторий/файл') .then(response => response.json()) .then(data => { console.log(data); }) .catch(error => console.error(error)); 4. Кодты интеграциялау • Wix сайтындағы элементтерге оқиға (events) немесе әрекеттер (actions) жаз: $w("#button1").onClick(() => { console.log("Батырма басылды!"); }); 5. Сайттың жұмысын тексеру • Барлық жазылған кодтардың дұрыстығын тексер. • Wix-тің Preview функциясын қолдана отырып, сайт жұмысының қалай көрінетінін көр.
